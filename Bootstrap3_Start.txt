# Bootstrap3_Start
A start template to build a website with Bootstrap 3 and PHP

Through PHP we can include parts of the site which repeated in several pages, like top, main-menu, footer and sidebar.

Folder Structure:

bootstrap3/
└── _assets/
│   ├── css/
│   │   └── custom.css
│   ├── fonts/
│   ├── img/
│   ├── js/
│   ├── modules/
│   │   ├── config.php
│   │   ├── footer.php
│   │   ├── ga.php
│   │   ├── header-calls.php
│   │   ├── scripts.php
│   │   └── top.php
└── index.php
